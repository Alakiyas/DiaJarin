Tambah Materi Baru

Materi akan muncul di sidebar sebagai kategori terpisah
Kosongkan untuk otomatis dari judul
Gunakan tag HTML: h2 untuk sub judul, p untuk paragraf, div class="code-block" untuk kode, div class="info-box" untuk tips

Preview Format Penulisan

<h2>Judul Section</h2> → Untuk judul besar dalam materi
<h3>Sub Judul</h3> → Untuk judul kecil / sub section
<div class="code-block"><pre><code class="language-php">...</code></pre></div> → Untuk menampilkan kode program
<div class="info-box"><strong>Tips:</strong> isi tips di sini</div> → Untuk menampilkan kotak informasi/tips
<ul><li>item 1</li><li>item 2</li></ul> → Untuk daftar bullet point
