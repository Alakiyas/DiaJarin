Login & Authorization

Praktik · 2026-05-14

Membangun sistem autentikasi dasar agar hanya pengguna yang sudah login yang bisa mengakses halaman tertentu.

Struktur Database

CREATE TABLE t_login (
    id_login INT(11) AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(100),
    password VARCHAR(100),
    level TINYINT(1)
);

INSERT INTO t_login (username, password, level)
VALUES ('admin', SHA1('admin'), 1);

Logika Login

<?php
session_start();
$sql = "SELECT * FROM t_login

Searching & Sorting

Praktik · 2026-05-15

Menambahkan fitur pencarian data dan pengurutan kolom dengan klik panah.

Form Pencarian

<form method="GET">
    <input type="text" name="search" placeholder="Cari NIS atau Nama">
    <button type="submit">Cari</button>
</form>

<?php
$search = @$_GET['search'];
if (!empty($search)) {
    $sql .= " WHERE nis LIKE '%$search%'
              OR nama_lengkap LIKE '%$search%' ";
}
?>

Tombol Sorting

<th>
    NIS
    <a href="?sort=nis&order=asc">▲</a>
    <a href="?sort=nis&order=desc">▼</a>
</th>
Penjelasan: Tanda % di LIKE berarti "mengandung kata tersebut di posisi manapun".

Materi 5

Relasi 2 Tabel

Praktik · 2026-05-18

Menghubungkan tabel siswa dengan tabel kelas menggunakan foreign key dan JOIN query.

Membuat Tabel Kelas

CREATE TABLE kelas (
    id_kelas INT(11) AUTO_INCREMENT PRIMARY KEY,
    nama_kelas VARCHAR(15),
    jurusan VARCHAR(50)
);

ALTER TABLE t_siswa CHANGE kelas id_kelas INT(11);

Query dengan INNER JOIN

SELECT * FROM siswa 
INNER JOIN kelas 
ON siswa.id_kelas = kelas.id_kelas;
Konsep: Relasi tabel memungkinkan data terstruktur dan tidak redundan.
