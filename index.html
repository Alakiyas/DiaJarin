MATERI TAMBAHAN

    Fun Fact

    Beberapa fakta menarik tentang PHP dan Laravel yang mungkin belum kamu tahu

    PHP Native

    1

    PHP pertama kali dibuat tahun 1994 oleh Rasmus Lerdorf sebagai kumpulan script CGI

    2

    Kepanjangan PHP awalnya adalah Personal Home Page, sekarang menjadi PHP: Hypertext Preprocessor

    3

    Sekitar 77% website di dunia menggunakan PHP sebagai backend-nya

    4

    WordPress, Facebook, dan Wikipedia awalnya dibangun dengan PHP

    Laravel

    1

    Laravel pertama kali dirilis tahun 2011 oleh Taylor Otwell

    2

    Nama "Laravel" terinspirasi dari kata "Laravel" yang artinya "istana" dalam bahasa Italia

    3

    Laravel adalah framework PHP dengan jumlah bintang terbanyak di GitHub

    4

    Fitur Eloquent ORM membuat interaksi database jadi seperti bicara dengan objek
