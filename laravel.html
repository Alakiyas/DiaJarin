Pengenalan

Apa Itu Laravel?

Teori · 2026-05-03

Laravel adalah PHP framework open-source yang mengikuti pola arsitektur MVC (Model-View-Controller). Dibuat oleh Taylor Otwell dan pertama kali dirilis pada tahun 2011.

Fitur Unggulan

  • Artisan CLI - Command line tools untuk mempercepat development
  • Eloquent ORM - Interaksi database dengan gaya OOP
  • Blade Template - Template engine yang powerful
  • Migration - Version control untuk database
MVC Pattern: Model (Data), View (Tampilan), Controller (Logika) - memisahkan kode agar lebih terstruktur.
Materi 1

Membuat Project Laravel

Instalasi · 2026-05-07

Instalasi Laravel sangat mudah dengan Composer.

# Buat project baru
composer create-project laravel/laravel nama-project

# Masuk ke folder project
cd nama-project

# Jalankan server development
php artisan serve

# Server akan berjalan di http://localhost:8000
Persiapan: Pastikan PHP 8.2+ dan Composer sudah terinstall.
Materi 2

Database Migration

Praktik · 2026-04-09

Migration seperti Git untuk database - melacak perubahan struktur database.

# Membuat migration
php artisan make:migration create_t_siswa_table

# Menjalankan migration
php artisan migrate

# Rollback migration terakhir
php artisan migrate:rollback
public function up()
{
    Schema::create('t_siswa', function (Blueprint $table) {
        $table->id();
        $table->integer('nis')->unique();
        $table->string('nama_lengkap', 100);
        $table->enum('jenis_kelamin', ['L', 'P']);
        $table->timestamps();
    });
}
Materi 3

Controller & Facade DB

Praktik · 2026-04-12

Controller adalah jembatan antara Model dan View. Facade DB memudahkan interaksi dengan database.

# Membuat controller
php artisan make:controller SiswaController
use Illuminate\Support\Facades\DB;

class SiswaController extends Controller
{
    public function index()
    {
        $siswa = DB::table('t_siswa')->get();
        return view('siswa.index', compact('siswa'));
    }
}
Materi 4

Create Data

Praktik · 2026-04-16

Menambahkan data baru ke database. Jangan lupa CSRF protection!

// Di route web.php
Route::post('/siswa', [SiswaController::class, 'store']);

// Di Controller
public function store(Request $request)
{
    $request->validate([
        'nis' => 'required|numeric',
        'nama_lengkap' => 'required|string'
    ]);
    
    DB::table('t_siswa')->insert($request->all());
    return redirect('/siswa')->with('success', 'Data berhasil ditambahkan');
}
Penting: Jangan lupa tambahkan @csrf di dalam form Laravel.

Materi 6

Eloquent ORM

Advanced · 2026-04-30

Eloquent membuat kode database kamu jadi lebih elegant. Setiap tabel punya Model sendiri.

// Membuat Model
php artisan make:model Siswa

// app/Models/Siswa.php
class Siswa extends Model
{
    protected $table = 't_siswa';
    protected $fillable = ['nis', 'nama_lengkap', 'jk'];
}

// Menggunakan Eloquent
$siswa = Siswa::all();                    // SELECT semua
$siswa = Siswa::find($id);               // Cari by ID
$siswa = Siswa::where('jk', 'L')->get(); // Filter
Siswa::create($request->all());          // Insert
$siswa->delete();                         // Delete
Keuntungan Eloquent: Kode lebih bersih, readable, dan mudah dimaintain.
